![Program](<images/program.jpeg>)

- 🔹 Yapay zeka nedir ve günümüzdeki yeri
- 🔹 Bilgisayarlar dünyayı nasıl algılar
- 🔹 Üretken yapay zeka araçları

- 🔹 Bulut ve IoT içinde yapay zeka - Robotik uygulamaları
